# 주기율표 학습 사이트 (Periodic Table Study)

화학 원소를 학습하고 탐색할 수 있는 인터랙티브한 주기율표 웹 애플리케이션입니다.

## 주요 기능

- **인터랙티브 주기율표**: 모든 118개 원소를 시각적으로 표현
- **원소 상세 정보**: 원소를 클릭하여 상세 정보 확인
  - 원자 번호, 기호, 원자량
  - 원소 분류 (금속, 비금속, 준금속 등)
  - 상태, 밀도, 녹는점, 끓는점
  - 전자 배치
  - 발견자 정보
- **원소 정보 편집**: 원소 정보를 편집하고 저장 가능 (localStorage에 저장)
- **검색 기능**: 원소 이름, 기호, 번호로 검색
- **반응형 디자인**: 모바일과 데스크톱 모두 지원
- **다크 모드**: 눈에 편안한 다크 테마

## 기술 스택

- **React 18** - UI 라이브러리
- **TypeScript** - 타입 안정성
- **Vite** - 빌드 도구
- **Tailwind CSS** - 스타일링
- **Lucide React** - 아이콘

## 설치 및 실행

### 필수 요구사항

- Node.js 16.x 이상
- npm 또는 yarn

### 설치

```bash
npm install
```

### 개발 서버 실행

```bash
npm run dev
```

개발 서버가 시작되면 브라우저에서 `http://localhost:5173`으로 접속하세요.

### 빌드

```bash
npm run build
```

### 미리보기

```bash
npm run preview
```

## 사용 방법

1. **원소 탐색**: 주기율표에서 원하는 원소를 클릭합니다.
2. **상세 정보 확인**: 클릭한 원소의 상세 정보가 모달로 표시됩니다.
3. **정보 편집**: 상세 정보 모달에서 편집 버튼을 클릭하여 원소 정보를 수정할 수 있습니다.
4. **검색**: 상단 검색창에 원소 이름, 기호, 또는 번호를 입력하여 원소를 찾을 수 있습니다.
5. **편집 내용 초기화**: 편집한 원소가 있으면 상단에 초기화 버튼이 표시됩니다.
